# H2petrons Website

Static multi-page site for H2petrons project. Built with TailwindCSS, no build step required.

## Pages
- index.html (Home)
- about.html (About the company)
- plans.html (Company plans & roadmap)
- tools.html (Tools & infrastructure)
- contact.html (Contact form & links)

## Free Hosting (GitHub Pages)
1. Create a public repo on GitHub (e.g., H2).
2. Upload these files to the repo root.
3. Go to Settings → Pages → Deploy from branch (main / root).
4. Site will be live at: https://<USERNAME>.github.io/H2/

## Customize
- Edit `index.html` and other pages with your text and images.
- Replace social media links in footer with your real accounts.
